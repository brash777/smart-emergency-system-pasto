# Problem Definition

In the city of Pasto, emergency response times can be delayed due to
lack of coordination and manual reporting systems.

Citizens must call emergency services, which may cause delays in
critical situations such as accidents, fires, or robberies.

This project proposes an intelligent system that allows users to report
emergencies using images and location data.

The system will use artificial intelligence to automatically identify
the type of emergency and assign the appropriate response service.

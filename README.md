# Smart Emergency System - Pasto

## Descripción

Este proyecto consiste en el desarrollo de una aplicación web que permite
reportar emergencias mediante imágenes y ubicación geográfica.

El sistema utiliza inteligencia artificial para identificar el tipo de
emergencia y asignar automáticamente el servicio correspondiente.

## Objetivos

• Reducir el tiempo de respuesta ante emergencias
• Automatizar la clasificación de incidentes
• Optimizar la asignación de recursos (policía, ambulancias, bomberos)

## Tecnologías

Backend: Java Spring Boot
Frontend: HTML, CSS, JavaScript
Base de datos: PostgreSQL
Inteligencia Artificial: Computer Vision
